# 🌐 Buddy Script - Full-Stack Social Networking Platform

A modern, high-performance social media application built with the **MERN** stack (MongoDB, Express, React, Node.js). This project features a minimalist modern social media UI, real-time data fetching, and a seamless user experience.

---

## 🚀 Key Features

* **Infinite Scrolling:** Implemented using `TanStack Query` (React Query) and `react-intersection-observer` for smooth, lag-free feed browsing.
* **Dynamic Post Management:** Users can create, view, and manage posts with real-time updates.
* **Secure Authentication:** Integrated with Firebase and JWT (JSON Web Tokens) for protected private routes and secure API communication.
* **Privacy Controls:** Smart filtering logic to display public posts and user-specific private content.
* **Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop using **Bootstrap 5** and custom CSS.
* **Aesthetic UX:** Features neon-themed error pages, skeleton loaders, and modern dark-mode aesthetics.

## 🛠️ Tech Stack

### Frontend
* **Framework:** React.js (Vite)
* **State Management:** TanStack Query (React Query)
* **Styling:** Bootstrap 5 & Custom CSS
* **HTTP Client:** Axios (with Interceptors for security)

### Backend
* **Runtime:** Node.js
* **Framework:** Express.js
* **Database:** MongoDB (with Mongoose)
* **Security:** JWT, CORS, Dotenv
